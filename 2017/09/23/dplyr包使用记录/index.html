分组动作 group_by()

以上5个动词函数已经很方便了， 但是当它们跟分组操作这个概念结合起来时，那才叫真正的强大！ 当对数据集通过group_by() 添加了分组信息后，mutate()、 arrange()和summarise()函数会自动对这些tbl类数据执行分组操作 (R语言泛型函数的优势)。例如：对飞机航班数据按飞机编号(TailNum)进行分组，计算该飞机航班的次数 (count = n())，平均飞行距离(dist = mean(Distance, na.rm = TRUE))和延时(delay = mean(ArrDelay, na.rm = TRUE))。
